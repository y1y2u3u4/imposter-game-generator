# Imposter Game Generator - Product Requirements Document

## 1. Product Overview

### 1.1 Product Name
**Imposter Game Generator** (impostergamegenerators.com)

### 1.2 Product Description
A free online party word game generator that creates word pairs for playing the "Imposter" (卧底词) game. Players receive secret words, with most getting the same word while one or more "imposters" receive a similar but different word. Through discussion and voting, players try to identify the imposters.

### 1.3 Target Users
- Party game enthusiasts
- Family gatherings
- Team building facilitators
- ESL teachers and students
- Friend groups looking for fun activities

### 1.4 Core Value Proposition
- **Instant Generation**: Get word pairs immediately without registration
- **Category Selection**: Choose from multiple themed categories
- **Customizable**: Adjust player count and imposter count
- **Mobile-Friendly**: Works seamlessly on all devices
- **Free to Use**: No paywalls or premium features

## 3. MVP Features

### F1: Word Pair Generator
- Generate word pairs from selected category
- Support 3-20 players
- Support 1-3 imposters
- Individual player cards with reveal/hide

### F2: Category Selection
- Animals, Food & Drinks, Everyday Objects, Places, Actions, Random
- Minimum 30+ word pairs per category

### F3: Player Card Display
- Tap to reveal word
- Clear visual design
- Mobile-friendly

### F4: Game Controls
- New Game, Shuffle Players, Reveal All, Share Game
